# Terminal Rules

黃金法則
=======
1. [不要做學徒](不要在那跪拜誰是誰偶像，那沒有意義)
2. [不要當守門](守門是指有人自行決定有權怎樣，或是身份)
3. [不要自大狂](永遠要謙虛，沒人喜歡自大的傻瓜)
4. [保持學生心](不管其他成員水平如何，都應對待其他成員一同學習)

- 通常規則
<不要假冒工作人員 = 這樣做將使您立即被"禁止在本社區活動"。>
<使用事實而非權威 = 我們應當養成追求真實，而"非紙上談兵"。>
<勿煽動語言或技術 = 如文本編輯器，有人喜歡"Vim"也有人喜歡"Notepad++">
<不要做廣告亂頻等 = 這與社區頻道無關，"也會讓你找不到同好"。>
<不要要求回應工作 = 用提問的方式來解決自己"應當"該做的事，是不行的。>
<不要參加湯匙餵食 = 比起給一人一魚，"不如教導釣魚"，你要餵養一輩子嘛?>
<沒有後座駕駛火車 = 看到有人在解決問題，請別接著瘋狂回應，"適當補充為佳"。>
<自由連結討論指南 = 當初學者不了解問題的情況，請多用連結過去嘗試了解"FAQ">

* 嚴厲規則
不準討論駭客的相關話題，通常會導致更多非法話題。
允許輕描淡寫地談論預防，不可以任何方式提供細節..
